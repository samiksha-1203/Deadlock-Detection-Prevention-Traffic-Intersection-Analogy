ℹ️ How It Works

Model Mapping:

  • 🚗 Cars = Processes (P1, P2, P3, P4)
  • 🛣️ Lanes = Resources (R_North, R_South, R_East, R_West) - Single instance each
  • ⚡ Each car needs 2 resources: Entry lane + Exit lane to complete journey

Three Modes:

  • 🔴 Detection Mode: Allows all requests, then detects deadlock using RAG cycle detection
  • 🟢 Avoidance Mode (Banker's): Checks if granting request keeps system in safe state before allowing
  • 🟡 Prevention Mode: Uses resource ordering (N→E→S→W) to prevent circular wait

Key Concepts:

  • Max Matrix: Declared upfront - each car declares its full resource need (entry + exit lane)
  • Safe State: A state where there exists a sequence to complete all processes
  • Resource Ordering: If all processes request resources in the same order, no circular wait can form
